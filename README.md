# account-growth-framework

Templates + frameworks for retention, renewals, and expansion (AM/CSM-ready).

## What’s inside
- **templates/** – account plan + stakeholder map
- **renewals/** – renewal motion playbook + risk signals
- **expansion/** – expansion paths, signals, and discovery prompts
- **qbr/** – exec-friendly QBR structure

## How to use this
1. Start with **templates/account-plan-template.md**
2. Run the renewal timeline in **renewals/renewal-motion-playbook.md**
3. Use **expansion/expansion-strategy.md** to build a growth plan
4. Wrap it all into a narrative using **qbr/qbr-structure.md**

*All examples are mock / generalized for public sharing.*
